## Example
**Question**
Find the value of \[8\cos^210^\circ - \dfrac{1}{\sin 10^\circ}.\]

**Solution**
```python
angle = math.radians(10)
value = 8 * math.cos(angle)**2 - 1 / math.sin(angle)
print(value)
```
**Tools**
```python
import math
```
